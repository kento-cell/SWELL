# Swell — デザイン計画書

## コンセプト
「ニュースを読む前に、波を見る」  
話題の温度感を「波」で可視化するウィジェットファーストのニュースアプリ。  
ピクセル調UIで独自の世界観を表現する。

## 画面一覧

### 1. ホーム画面（メイン）
- **役割**: 1ページ1トピック表示。左右スワイプで前後移動。
- **コンテンツ**:
  - 上部: カテゴリタブ（NEWS / SOCIAL / MARKET）
  - 中央: トピックカード（タイトル・要約・波アニメーション）
  - 下部: ページインジケーター・ソースバッジ
- **インタラクション**:
  - 左右スワイプ or 矢印ボタンでトピック移動
  - タップでトピック詳細へ遷移
  - SOCIAL/MARKETはFreeでロック表示

### 2. トピック詳細画面
- **役割**: 選択したトピックの詳細情報を表示
- **コンテンツ**:
  - タイトル・詳細要約
  - 波の大きさ・色の説明
  - ソース名・元記事リンク
  - 関連トピック（将来）
- **インタラクション**:
  - 戻るボタン
  - 元記事を開く（外部ブラウザ）

### 3. 設定画面
- **役割**: プラン確認・テーマ切替・アプリ情報
- **コンテンツ**:
  - 現在のプラン（Free/Premium）
  - Premiumアップグレードボタン
  - テーマ（ダーク/ライト）
  - バージョン情報

### 4. Premiumアップグレードシート
- **役割**: Premium機能の訴求・購入導線
- **コンテンツ**:
  - Free vs Premium 比較
  - 月額480円の表示
  - 購入ボタン（モック）

### 5. 初回チュートリアル（オーバーレイ）
- **役割**: 初回起動時のみ表示。波の見方・操作方法を説明。
- **コンテンツ**:
  - ステップ1: 波の色の意味
  - ステップ2: 波の高さの意味
  - ステップ3: 左右移動の操作
  - ステップ4: カテゴリの説明

## キーユーザーフロー

### フロー1: トピック閲覧
1. ホーム画面（NEWSタブ）でトピックを確認
2. 左右スワイプで次のトピックへ移動
3. 気になるトピックをタップ → 詳細画面へ
4. 元記事リンクをタップ → 外部ブラウザで開く
5. 戻るボタンでホームへ

### フロー2: Premium体験
1. SOCIALタブをタップ → ロック表示
2. 南京錠アイコンをタップ → Premiumシート表示
3. 購入ボタン（モック）

### フロー3: 初回チュートリアル
1. 初回起動 → チュートリアルオーバーレイ表示
2. 4ステップを順に読む
3. 「はじめる」ボタンでホームへ

## ピクセルUIコンポーネント

### 波コンポーネント（WaveDisplay）
- SVGベースのピクセル風波形アニメーション
- 高さ: low / medium / high
- 色: blue / green / yellow / red

### ピクセル矢印（PixelArrow）
- 左右三角のピクセル表現
- ロック時は南京錠アイコンを重ねる

### ソースバッジ（SourceBadge）
- ニュースソース名を小さく表示

### ロックオーバーレイ（LockOverlay）
- SOCIAL/MARKETのロック表示
- 半透明オーバーレイ + 南京錠アイコン
