## 推奨：Finnhub を採用

### 理由：
1. **高いレート制限:** 60 calls/分 = 月 2.5M リクエスト
2. **実用的:** SWELL のような中規模アプリに十分
3. **登録簡単:** メールアドレスのみで登録可能
4. **無料永久:** Personal Use ライセンスで無料

### 実装方法：
```typescript
// Finnhub API キー取得
// https://finnhub.io/ で登録 → API Key 取得

const FINNHUB_API_KEY = 'your_api_key_here';

// 株価取得
async function getStockPrice(symbol: string) {
  const response = await fetch(
    `https://finnhub.io/api/v1/quote?symbol=${symbol}&token=${FINNHUB_API_KEY}`
  );
  const data = await response.json();
  return {
    price: data.c,
    change: data.d,
    changePercent: data.dp,
    high: data.h,
    low: data.l,
  };
}
```
