# SWELL — DEVELOPMENT PLAN

> **このファイルは人間（開発者）が編集するロードマップです。**
> AIエージェントはこのファイルを読んで次のタスクを判断します。
> タスクを追加・変更・優先度変更する場合はこのファイルを直接編集してください。

---

## 使い方

各タスクの先頭の記号を変更することで状態を管理します。

| 記号 | 意味 |
|------|------|
| `[ ]` | 未着手（AIが次に実装する） |
| `[>]` | 進行中 |
| `[x]` | 完了 |
| `[-]` | 保留・スキップ |
| `[!]` | 緊急バグ修正 |

## メモ・制約事項

**コスト制約:** ランニングコスト完全ゼロが絶対条件。LLM・有料API・クラウドサービスは使用禁止。YouTube API は無料枠（1日10,000クォータ）内で運用。

**技術制約:** Expo SDK 54・React Native 0.81 の範囲内で実装。ネイティブモジュールの追加は最小限に。

**デザイン制約:** 全コンポーネントは `useThemeContext()` でテーマカラーを参照。ハードコードされた色は禁止。
